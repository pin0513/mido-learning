# SEO Audit Skill

## Meta Tags 檢查

### Title Tag
- [ ] 存在且唯一
- [ ] 長度 < 60 字元
- [ ] 包含主要關鍵字
- [ ] 格式：{主要內容} | {品牌名稱}

### Meta Description
- [ ] 存在且唯一
- [ ] 長度 150-160 字元
- [ ] 包含關鍵字
- [ ] 具吸引力（引導點擊）

### Open Graph Tags
- [ ] og:title
- [ ] og:description
- [ ] og:image
- [ ] og:url
- [ ] og:type

---

## 內容架構檢查

### 標題層級
- [ ] 每頁只有一個 H1
- [ ] H1 包含主要關鍵字
- [ ] 標題層級正確（H1 > H2 > H3，不跳級）
- [ ] 標題清楚描述內容

### 內容長度
- [ ] 首頁 ≥ 300 字
- [ ] 服務頁 ≥ 500 字
- [ ] 部落格 ≥ 800 字

### 內部連結
- [ ] 每頁至少 3-5 個內部連結
- [ ] 連結文字描述清楚（不使用「點這裡」）
- [ ] 無失效連結

---

## 結構化資料檢查

### Organization Schema
- [ ] 已實作
- [ ] 通過 Google Rich Results Test

### Breadcrumb Schema
- [ ] 已實作（若適用）

### Article Schema
- [ ] 已實作（部落格文章）

---

## 圖片 SEO 檢查

### Alt Text
- [ ] 所有圖片有 alt text
- [ ] Alt text 描述清楚
- [ ] 包含關鍵字（自然使用）

### 檔案名稱
- [ ] 描述性檔案名稱
- [ ] 使用連字號分隔
- [ ] 避免中文

### 圖片優化
- [ ] 使用 WebP 格式
- [ ] 檔案大小 < 200KB
- [ ] 實作 lazy loading

---

## 技術 SEO 檢查

### Robots.txt
- [ ] 存在
- [ ] 格式正確
- [ ] 包含 Sitemap 連結

### Sitemap.xml
- [ ] 存在
- [ ] 格式正確
- [ ] 包含所有重要頁面

### Canonical Tags
- [ ] 已實作（避免重複內容）

### Hreflang Tags
- [ ] 已實作（多語言網站）

---

## Core Web Vitals 檢查

- [ ] LCP < 2.5s
- [ ] FID < 100ms
- [ ] CLS < 0.1

---

## Mobile SEO 檢查

- [ ] 通過 Mobile-Friendly Test
- [ ] Viewport meta tag 存在
- [ ] 文字大小適當（≥ 16px）
- [ ] 觸控目標足夠大（≥ 44x44px）
- [ ] 載入速度 < 3s

---

## 檢查工具

- Google PageSpeed Insights
- Google Rich Results Test
- Google Mobile-Friendly Test
- W3C HTML Validator
- WebAIM Contrast Checker
