# Design Review Skill

## 視覺層次檢查

- [ ] 標題、內文、輔助文字大小對比清晰
- [ ] 重要元素（CTA 按鈕、主要訊息）突出
- [ ] 視覺動線引導使用者關注核心內容
- [ ] 字重對比明確（Bold vs Regular）

---

## 色彩運用檢查

- [ ] 品牌主色正確使用
- [ ] 色彩搭配和諧
- [ ] 對比度足夠（WCAG AA：文字 4.5:1、UI 3:1）
- [ ] 色彩使用有意義（藍色=可信賴、紅色=警示）
- [ ] 色彩不超過 3-4 種主要顏色

---

## 字體排版檢查

- [ ] 字體選擇符合品牌調性
- [ ] 字級大小合理（標題 24-48px、內文 16-18px）
- [ ] 行距舒適（1.5-1.8）
- [ ] 字重對比明確
- [ ] 行長控制在 60-80 字元

---

## 空間與節奏檢查

- [ ] 元素之間間距一致
- [ ] 使用統一的間距系統（8px 基準）
- [ ] 內容區塊之間有足夠呼吸空間
- [ ] 整體佈局平衡（不頭重腳輕或左右失衡）

---

## 視覺細節檢查

- [ ] 圖標風格統一（全部 outline 或全部 filled）
- [ ] 圓角半徑一致（按鈕 8px、卡片 12px）
- [ ] 陰影合理（不過度使用，符合光源邏輯）
- [ ] 邊框粗細統一
- [ ] 所有按鈕有 hover 狀態

---

## 品牌一致性檢查

- [ ] Logo 使用正確（大小、位置、安全距離）
- [ ] 品牌色彩遵循品牌指南
- [ ] 視覺風格符合品牌調性
- [ ] Header 與 Footer 在所有頁面一致
- [ ] 按鈕樣式在所有頁面統一

---

## 使用者體驗檢查

- [ ] 文字與背景對比足夠
- [ ] 行長不過長（60-80 字元）
- [ ] 段落之間有足夠間距
- [ ] 按鈕明顯可點擊
- [ ] 重要訊息容易找到

---

## 審核結果

- [ ] ✅ 通過（可進入客戶確認）
- [ ] 🔄 需修正（必須修正項目完成後重新送審）
- [ ] ❌ 重大問題（需重新設計）

---

**使用此 skill 可確保設計審核的系統化與一致性。**
