什麼是 Python？

Python 是一種高階程式語言，具有以下特點：

  • 簡潔易讀 - 語法接近自然語言
  • 功能強大 - 從網頁到人工智慧
  • 社群活躍 - 豐富的函式庫資源
  • 跨平台 - Windows、Mac、Linux

Python 廣泛應用於：資料科學、機器學習、網頁開發、自動化腳本等領域。

變數與資料型態

變數是用來儲存資料的容器：

# 數字
age = 25
price = 99.9

# 文字
name = "小明"

# 布林值
is_student = True

# 列表
fruits = ["蘋果", "香蕉", "橘子"]

條件判斷 if-else

根據條件執行不同的程式碼：

score = 85

if score >= 90:
    print("優秀！")
elif score >= 60:
    print("及格")
else:
    print("需要加油")

執行結果：及格

迴圈 for & while

重複執行程式碼的方法：

# for 迴圈
for i in range(5):
    print(i) # 輸出 0,1,2,3,4

# while 迴圈
count = 0
while count < 3:
    print("Hello!")
    count += 1
